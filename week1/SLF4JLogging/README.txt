Exercise 1: Logging Error Messages and Warning Levels using SLF4J

Objective:
To demonstrate logging using the SLF4J logging framework.

Files Included:
- LoggingExample.java
- pom.xml

Dependencies:
- slf4j-api
- logback-classic

Logging Levels Demonstrated:
1. ERROR
2. WARN

Outcome:
This exercise demonstrates how SLF4J can be used to generate error and warning log messages in a Java application.
