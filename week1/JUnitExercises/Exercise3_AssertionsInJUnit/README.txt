Exercise 3: Assertions in JUnit

Objective:
To understand and use different JUnit assertion methods for validating test results.

Assertions Demonstrated:
1. assertEquals()
2. assertTrue()
3. assertFalse()
4. assertNull()
5. assertNotNull()

Files Included:
- AssertionsTest.java
- pom.xml

Outcome:
This exercise demonstrates how JUnit assertions are used to verify expected behavior and validate program correctness during unit testing.
