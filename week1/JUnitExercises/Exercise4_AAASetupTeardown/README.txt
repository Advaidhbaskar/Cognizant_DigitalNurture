Exercise 4: Arrange-Act-Assert (AAA) Pattern, Setup and Teardown Methods

Objective:
To understand the Arrange-Act-Assert testing pattern and the use of setup and teardown methods in JUnit.

Concepts Covered:
1. Arrange-Act-Assert (AAA) Pattern
2. @Before Annotation
3. @After Annotation
4. Test Fixtures

Files Included:
- Calculator.java
- CalculatorAAATest.java
- pom.xml

Outcome:
This exercise demonstrates how to organize unit tests using the AAA pattern and how setup and teardown methods help prepare and clean up test environments before and after test execution.
